# VIBECODE_IDE
VIBECODE_IDE_GUI


The settings allow you to configure the AI, language, and window mode.

I personally recommend using a local AI setup, since this tool is originally aimed at users who do not want to pay for subscriptions and want to save tokens.

You can click COPY to copy the generated prompt, allowing the AI to carry out precise boundary and requirement discussions.

The final specification document can then be pasted into a new conversation to generate code.
